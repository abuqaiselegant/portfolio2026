Stock market prediction model using LASSO Regression, Technical Indicators, and Time Lag. Implemented for NSE NIFTY50.

Key features: LASSO for feature selection and regularization, technical indicators (Moving Averages, RSI, Super Trend), time lag calculations for sequential patterns, expanded feature creation from limited initial data points.

Results: Test RMSE 14.78, Train RMSE 13.4, Test MAPE 2.98%. Published in IEEE.
