Lightweight ML experiment tracking platform for engineers who want structured logging without the overhead of MLflow.

Experiments are logged via a 4-line Python snippet or REST API with no extra dependencies. The dashboard visualises metrics and training curves in real time and supports model upload and download for artefact management.

FastAPI and PostgreSQL backend, Next.js 16 with TypeScript and shadcn/ui frontend, deployed on Vercel and Render.
