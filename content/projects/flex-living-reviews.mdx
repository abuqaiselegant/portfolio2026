A modern review management system for vacation rental properties with elegant public review display and intuitive manager approval dashboard.

Public interface: browse properties with approved review counts, view detailed reviews with category ratings, mobile-optimized design. Manager dashboard: review approval workflow (approve/reject/pending), real-time approval statistics, KPI metrics and analytics, property performance insights.

PostgreSQL-only architecture on AWS RDS, Next.js API Routes on Vercel, GitHub CI/CD for automatic deployment. Built with Next.js 14, TypeScript, and Tailwind CSS.
