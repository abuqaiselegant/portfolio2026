Review management system for vacation rental properties, with a public-facing display and a manager approval dashboard.

The public interface lets guests browse properties with aggregated ratings and per-category review breakdowns on a mobile-optimised layout. The manager dashboard handles approve/reject/pending workflows, tracks real-time approval stats, and surfaces property-level performance metrics.

Built with Next.js 14, TypeScript, and Tailwind CSS, backed by PostgreSQL on AWS RDS. Deployed on Vercel with GitHub Actions CI/CD.
