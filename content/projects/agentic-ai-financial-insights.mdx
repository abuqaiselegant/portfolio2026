Designed an agentic AI system using LangGraph to orchestrate multi-step reasoning, retrieval workflows, and memory-augmented behaviour.

Integrated a VectorDB-driven RAG layer with session-aware FastAPI endpoints, enabling personalised outputs for 50+ users. Deployed on AWS EC2 with Docker and CI/CD, achieving ~1.4s median latency under concurrent load.

Implemented secure authentication, request tracing, and structured logging for reliable, production-ready operation.
