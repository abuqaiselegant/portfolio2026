Building language models from scratch—no magic, no hand-waving. Understand exactly what happens when a neural network trains, how attention works, and how GPT-style models are constructed.

Learning path covers: Neural Networks (NumPy-only NN, forward pass, backprop, PyTorch comparison), Attention (tokenization, embeddings, positional encoding, attention from scratch), GPT (Tiny Shakespeare implementation, modern LLM with RMSNorm, SwiGLU, RoPE, GQA), and Agents (API calls to agents with OpenRouter).

Includes an interactive Modern GPT Guide (React/JSX) for BPE tokenizer, embeddings, attention, and GPT architecture walkthrough.
