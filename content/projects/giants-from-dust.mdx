Open-source repository for building modern AI architectures from scratch — no black-box abstractions, just clean implementations with visible internals.

Covers the full stack from fundamentals to production-grade models: NumPy-only neural networks with manual backprop, attention built from tokenisation through positional encoding, a Tiny Shakespeare GPT, and a modern LLM with RMSNorm, SwiGLU, RoPE, and Grouped Query Attention. Ends with an agentic workflow using the OpenRouter API.

Includes an interactive React/JSX guide walking through BPE tokenisation, embeddings, attention, and GPT architecture step by step.
