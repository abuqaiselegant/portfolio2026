Built a modular sentiment-analysis pipeline integrating multiple LLMs and benchmarked 8+ model variants across accuracy, consistency, reasoning quality, and inference cost.

Designed structured evaluation prompts and scoring criteria to quantify sentiment drift and calibration differences between LLM models, enabling informed model selection for real-time and batch workflows. Combined LLM-derived sentiment with market indicators to generate trading signals, achieving a Sharpe ratio of 1.34 over 1.2K+ backtest days.

Deployed as a Flask API with configurable model-selection modes (high-accuracy / low-latency / cost-efficient).
