Built a distributed Spark pipeline using SMOTE variants and structured resampling to improve minority-class recall on 100K+ highly skewed records.

Optimised preprocessing and synthetic sample generation with partition tuning, caching, and broadcast joins, reducing training time by ~30% on 100K+ records. Designed a reproducible experiment framework with MLflow, comparing imbalance-mitigation techniques across 25+ tracked runs.

Deployed the model as a Dockerised service on Cloud Run, achieving ~1.2s inference latency under concurrent load.
