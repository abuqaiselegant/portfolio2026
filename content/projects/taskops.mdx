Task management system backend built with Flask, PostgreSQL, Redis and Docker. Containerised services ensure consistent behaviour across development and production environments.

Added a lightweight Node.js utility service for background checks and admin tasks (validation, log parsing), improving maintainability and reducing manual operations.

Implemented request validation, structured error handling and automated jobs that reduced debugging cycles by ~25% and improved system reliability. Documented API behaviour, configuration workflows and deployment steps to support predictable updates and smooth team handovers.
