Task management backend built with Flask, PostgreSQL, Redis, and Docker. Containerised services maintain consistent behaviour across development and production environments.

A lightweight Node.js utility service handles background jobs — input validation, log parsing, and admin operations — keeping the Flask layer focused on the core API.

Implemented JWT-based auth, request validation, and structured error handling. Automated background jobs reduced manual debugging cycles by ~25%.
